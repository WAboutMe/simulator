# simulator
实现小车基本功能：
WASD控制小车移动旋转，鼠标控制小车云台以及视野的俯仰与旋转；
鼠标左键发射子弹；
场地Buff：随机生成，得到后小车移动速度加成20%，子弹发射速度加成20%，持续时间8s，每15s刷新一次
